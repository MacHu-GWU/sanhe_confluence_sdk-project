Install
------------------------------------------------------------------------------

``sanhe_confluence_sdk`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install sanhe-confluence-sdk

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade sanhe-confluence-sdk
